#### Questions Type

# File Upload

The File Upload question type allows respondents to upload files related to your survey, such as production documents or requirement specifications. It features a title to guide the user on what to upload and an optional description to provide additional context.

# Configuration

## Options

1. **Title:** Add a clear title that informs the respondent about the purpose of the file upload, such as "Upload Production Documents."
2. **Description:** Provide an optional description to give respondents more details or instructions about what files they need to upload.
3. **Allow Multiple Files:** Enable this option to allow respondents to upload multiple files at once.
4. **Max File Size:** You can set a maximum file size limit, and an input box will appear to specify the size in MB.

<MdxImage
  src={FileUploadPreviewType}
  alt="Overview of File Uploade question type option max size"
  quality="100"
  className="max-w-full rounded-lg sm:max-w-3xl"
/>

5. **File Type Restrictions:** You can restrict the allowed file types. An input box will appear where you can specify the file formats, such as `.pdf`, `.jpg`, `.docx`, etc.
