#### Advanced Settings Under All Question Types

- **"Next" Button Lable** : This button label can be customized for each question type.
- **"Back" Button Lable** : This button label can be customized for each question type.

These labels are available for each question selected from question types.

#### Required Toogle

At the bottom right, there is a toggle. When enabled, respondents must explicitly answer this question. Without filling it out, they will not be able to proceed further.

#### Conditional Logic

With this option, you can set conditional logic based on the respondent's answers. Based on matching conditions, you can escalate or trigger specific actions. This option is available for each question, allowing for multiple conditional logic rules, making the process highly dynamic.
