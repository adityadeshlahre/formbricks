#### Question Type

# Date

The Date question type allows respondents to provide a date, such as when they are available or when an event is scheduled. It features a title to guide the respondent on what date to enter, and an optional description to provide further details or context.

/>

# Configuration

## Options

1. **Title:** Add a clear title to inform the respondent what date you are asking for, such as "Select Your Availability."
2. **Description:** Provide an optional description with further instructions, such as "Please select a date when you will be available for the meeting."
3. **Date Format:** Choose from multiple date formats for the input:
   - MM-DD-YYYY
   - DD-MM-YYYY
   - YYYY-MM-DD
