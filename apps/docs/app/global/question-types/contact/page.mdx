#### Question Type

# Contact Info

The Contact Info question type allows respondents to provide their basic contact information such as name, email, and phone number. You can customize the form with a title, an optional description, and control which fields to display and require.

# Configuration

## Options

1. **Title:** Specify a title to describe the information you're collecting (e.g., "Enter Your Contact Details").
2. **Description:** Optionally, add a description to give additional context (e.g., "Please provide your contact information for further communication.").
3. **Toggle Fields:** You can choose to show and require any or all of the following fields:
   - First Name
   - Last Name
   - Email
   - Phone Number
   - Company
